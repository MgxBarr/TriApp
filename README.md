# 🏁 Endurance Training Dashboard

Dashboard de performance pour triathlètes. Visualisation de la charge, planification de saison et moteur de construction de séances.

## 🚀 Récupération du projet

### 1. Cloner le dépôt
Récupérez le code source depuis GitHub :
```bash
git clone [https://github.com/MgxBarr/TriApp.git](https://github.com/MgxBarr/TriApp.git)
cd TriApp
```

### 2. Installer les dépendances
Cette commande installe automatiquement toutes les bibliothèques nécessaires (Next.js, Tailwind, Lucide, Recharts, Supabase, etc.) :
```bash
npm install
```

### 3. Configurer les variables d'environnement 
- Créé un fichier .env.local 
- Ajoute tes identifiants Supabase (que tu trouveras dans les réglages de ton projet Supabase > API) :
```bash
NEXT_PUBLIC_SUPABASE_URL=votre_url_projet_supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY=votre_cle_anonyme_supabase
```

### 4. Lancer l'application  
Démarre le serveur de développement local :
```bash
npm run dev
```
L'application sera accessible sur http://localhost:3000.
